MANAGER

Live Watcher

打开即扫描，前台看列表，后台看通知。

实时监听插入、拔出后快速刷新列表。
托盘常驻最小化后继续在后台工作。
低打扰通知窗口不在前台时才提示。
ALIAS

Custom Alias

给每个串口起一个你真正看得懂的名字。

COM7未命名默认只显示系统识别到的端口。
→
COM7网关 AP回车确认，之后可再次编辑。
→
COM7网关 AP插拔通知也优先显示这个名字。

EVENT

Background Watcher

前台不打扰，后台才通知。

窗口在前台时，变化直接更新在列表和最近事件里；工具在后台时，系统通知告诉你哪个串口发生变化。
